Discord Webhook Setup

Get alerts in any Discord channel whenever a stat updates.

1

Open Discord channel settings

Go to the channel you want alerts in → Settings → Integrations → Webhooks → New Webhook.

2

Copy the webhook URL

Click "Copy Webhook URL" — it looks like https://discord.com/api/webhooks/123.../abc...

3

Set environment variable

On Railway: go to your project → Variables → add DISCORD_WEBHOOK_URL with the URL as the value. Redeploy.

Sportradar API (Optional — for automatic live stats)

Without this, you update stats manually from the dashboard. With it, stats poll automatically every 60s.

1

Sign up at developer.sportradar.com

Free trial keys are available for NBA, NFL, MLB, NHL. Each sport has its own key.

2

Add your key

Set SPORTRADAR_API_KEY in your Railway environment variables. The poller will start automatically on next deploy.

3

Manual mode works great too

You can always update a player's current stat from the dashboard yourself — the Discord alert fires instantly when you hit Update.

Deploy to Railway (Free)

Get this running in the cloud in ~5 minutes.

1

Push to GitHub

Create a new GitHub repo and push this project folder to it.

2

Create Railway project

Go to railway.app → New Project → Deploy from GitHub → select your repo. Railway auto-detects Python.

3

Add environment variables

In Railway → Variables, add DISCORD_WEBHOOK_URL and optionally SPORTRADAR_API_KEY. Redeploy.

4

Get your public URL

Railway gives you a yourapp.railway.app URL. Open it, add your bets, and Discord alerts start flowing.
